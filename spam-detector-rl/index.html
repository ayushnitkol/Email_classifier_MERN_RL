How It Works

Our reinforcement learning agent uses Q-Learning to classify emails:

  • Feature Extraction: TF-IDF vectorization and keyword analysis
  • State Representation: Email features as environment states
  • Action Space: Classify as spam or ham
  • Reward System: +1 for correct classification, -1 for incorrect
  • Learning: Q-table updates based on classification rewards
